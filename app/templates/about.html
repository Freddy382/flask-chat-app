About


What is F-Chat?

F-Chat is a simple chat app made with the micro web framework Flask for the programming language Python
. It allows the user to communicate with other users trough individual chat rooms that he is able to create himself.
He can manage his created rooms at any time by adding a user, removing a user, clearing the room or completely deleting it.
F-Chat is Open Source and the Source Code can be found here.

How does F-Chat work?

Networking

In order to establish a connection between the different users and the server, F-Chat uses the python module Flask-SocketIO.
It allows many clients to connect to the server which is then going to handle the different clients and group them into rooms. The server-side of the app
is written in Python and the client-browser-side in JavaScript.
The clients and the server are able to communicate through so-called events which are triggered based on an action on the client side. When a user for example logs into the chat app, the join event on the client-side is triggered which it emits to the server-side using the Flask-SocketIO module. The server side
receives and handles it and then broadcasts it to the other users in the joined room.

Data Saving

All the data that the app needs are saved in a SQLite database which the server accesses and modifies using the python module Flask-SQLAlchemy. The database contains users, rooms and messages, which are accessed and changed during runtime.The passwords of all users are hashed and are therefore not visible for the admin.

Session Management

Data only relevant while the user is logged in is not saved in the Database. Instead, it is saved in the user session which is made with the Flask-Session python module It contains data about the current room the user is in as well as the username of the user which is used for rendering the HTML templates.

API

F-Chat offers a simple API that can be used by every user. It offers an easy way to analyze the app's data by querying the database.
In order to use it, you need to get an API token. A description on how you can get this token can be found below.

Endpoints


  • Get token: You need to pass your username as well as your password in json format
    /api/token

  • Get your hashed password: /api/me
